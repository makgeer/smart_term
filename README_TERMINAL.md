# Smart Term - Полнофункциональный кроссплатформенный терминал

## 🎯 Основная функция: ТЕРМИНАЛ

Smart Term - это **полнофункциональная командная строка** для всех операционных систем с дополнительными возможностями.

## ✅ Что работает (Основной функционал - ТЕРМИНАЛ):

### 💻 Терминал (ОСНОВНАЯ ФУНКЦИЯ)
- ✅ Полноценная командная строка
- ✅ История команд (стрелки вверх/вниз)
- ✅ Редактирование команд (Backspace, Delete, стрелки)
- ✅ Выполнение системных команд
- ✅ Встроенные команды: `help`, `clear`, `pwd`, `ls`, `cd`
- ✅ Промпт и курсор
- ✅ Псевдографический интерфейс

### 🎮 Управление терминалом
- **Enter** - выполнить команду
- **Стрелки вверх/вниз** - навигация по истории команд
- **Стрелки влево/вправо** - навигация по команде
- **Backspace/Delete** - удаление символов
- **Home/End** - переход к началу/концу строки
- **Tab** - автодополнение (показывает пробелы)

### 🔧 Дополнительные режимы (как дополнение к терминалу)
- **Tab** - переключение в файловый менеджер
- **Ctrl+G** - Git статус (если в Git репозитории)
- **F4** - редактор (в разработке)
- **Ctrl+P** - псевдографика (демонстрация)
- **Ctrl+Q** - выход

## 🚀 Быстрый старт

### Запуск
```bash
# Через run.bat
run.bat

# Или напрямую
target\release\smart-term.exe
```

### Основные команды терминала
```bash
help        # Справка по командам
clear       # Очистка экрана
pwd         # Текущая директория
ls          # Список файлов
cd ..       # Переход вверх
cd ~        # Переход в домашнюю директорию
dir         # Системная команда Windows
echo Hello  # Вывод текста
```

## 📝 Встроенные команды терминала

| Команда | Описание |
|---------|----------|
| `help` | Показать справку |
| `clear` | Очистить экран |
| `pwd` | Показать текущую директорию |
| `ls` | Список файлов |
| `cd <путь>` | Сменить директорию |
| `exit` | Информация о выходе |
| `gui` | Информация о GUI режиме |

## 🖥️ Системные команды

Терминал поддерживает выполнение **любых системных команд**:

```bash
# Windows команды
dir                    # Список файлов
type filename.txt      # Просмотр файла
echo Hello World       # Вывод текста
notepad               # Запуск блокнота
tasklist              # Список процессов
ipconfig              # Информация о сети

# Любые другие команды
python script.py       # Выполнение Python
cargo build           # Сборка Rust проекта
git status            # Git команды
```
